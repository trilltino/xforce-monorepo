# IDL Files

Solana Program Interface Definition Language (IDL) files for XForce Terminal.

## Files

- `batch_swap_router.json` - Batch Swap Router program IDL

## Purpose

IDL files describe the interface of Solana programs, including:
- Account structures
- Instruction parameters
- Event definitions
- Error codes

## Usage

Used by the client library to generate TypeScript/Rust bindings for program interaction.
