# Documentation

HTML documentation for XForce Terminal architecture and design.

## Files

- `index.html` - Main documentation page
- `architecture.html` - System architecture diagrams
- `tech-stack.html` - Technology stack overview
- `terminal.html` - Terminal application documentation
- `contracts.html` - Smart contract documentation
- `data-flows.html` - Data flow diagrams
- `news-service.html` - News service documentation
- `styles.css` - Shared stylesheet

## Viewing

Open any HTML file in a web browser:

```bash
start docs/index.html
```

## Updating

Documentation is static HTML. Edit files directly to update content.
