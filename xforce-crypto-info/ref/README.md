# Reference Implementations

Reference code and examples for technologies used in XForce Crypto Info.

## Directories

### axum/
Rust web framework examples and patterns.

### leptos/
Rust web framework examples for WASM frontend development.

### polars/
DataFrame library examples for data processing.

### pyo3/
Rust-Python bindings examples.

### sqlx/
Rust SQL toolkit examples for database operations.

## Purpose

These directories contain:
- Example code for learning
- Reference implementations
- Proof-of-concept experiments
- Technology evaluation code

## Note

Code in these directories may be incomplete or experimental. Do not use in production without review.
